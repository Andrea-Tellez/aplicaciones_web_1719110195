Elemento !DOCTYPE html:

Es una instrucción especial que va al inicio de nuestro documento HTML y que permite al navegador entender qué versión de HTML estamos utilizando. Esta información determinará la manera en la que el navegador procesará el documento.

Elemento html y /html:

Representa la raíz de un documento HTML o XHTML. Todos los demás elementos deben ser descendientes de este elemento.

Parámetros:

Manifest: Especifica la dirección URL de un manifiesto de recursos que indica los recursos que deben ser almacenados en la caché local.

Version: Especifica la versión Document Type Definition de HTML del documento actual.

Xmlns: Especifica el Espacio de nombres XML del documento.

Elemento head y /head:

Representa una colección de metadatos acerca del documento, incluyendo enlaces o definiciones de, scripts y hojas de estilo.

Parámetro:

Profile: Los URIs de uno o más perfiles de metadatos, separados por un espacio en blanco.

Elemento title y /title:

Indica el título del documento, se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada.

Parámetro:

Idioma: lang (información sobre el idioma), dir (dirección del texto)

Elemento body y /body:

Es el contenedor para todos los aspectos representables de un documento. En estelugar los autores arrojan todos los elementos y contenidos que sus visitantes alcanzarán.

Parámetros:

Alink: Es el color por defecto que todos los elementos a contenidos por el elemento tendrán cuando el puntero del mouse se encuentre sobre ellos.

Background: Es el URL de una imagen que será utilizada para cubrir el fondo del elemento.

Bgcolor:Es un color para rellenar el fondo del elemento.

Text: Es el color predeterminado para el texto normal dentro del elemento.

Link: Es el color predeterminado del texto en todos los vínculos dentro del elemento

Vlink:Es el color por defecto para los vínculos dentro del elemento, que ya han sido visitados.

Onafterprint: Función para llamar después de que el usuario haya impreso el documento.

Onbeforeprint: Es cuando el usuario ha requerido que el documento sea impreso. Este evento es disparado exactamente antes de que el documento se imprima y puede ser utilizado para confirmar, retrasar o cancelar el proceso.

Onbeforeunload: El documento está a punto de ser removido o cerrado. Este evento es disparado exactamente antes de que el documento se cierre y puede ser utilizado para confirmar o retrasar el proceso.

Onhashchange: La parte del URL que se encuentra después del símbolo numeral ("#") ha cambiado.

Onlanguagechange:Es cuando el lenguaje preferido del usuario ha cambiado.

Onmessage: Es cuando el documento actual ha recibido un mensaje.

Onmessageerror: Es cuando el documento actual ha recibido un mensaje pero este no ha podido ser descifrado.

Onoffline: Es cuando la conexión a la red se ha perdido.

Ononline: Es cuando la conexión a la red ha sido restablecida.

Onpagehide: Es cuando el documento actual ha sido removido del área de visibilidad durante un recorrido por el historial de la sesión (una navegación hacia atrás/adelante).

Onpageshow: Es cuando el documento ha sido mostrado durante un recorrido por el historial de la sesión (una navegación hacia atrás/adelante o una actualización).

Onpopstate: Es cuando ha ocurrido un cambio de documentos en el historial de la sesión. Este evento es comúnmente disparado por una navegación hacia atrás/adelante.

Onstorage: Es cuando la información en el área de almacenamiento web (Web Storage) ha cambiado.

Onunload: Es cuando el documento ha sido removido o cerrado.

Elemento h1 y /h1:

Representa un encabezado de nivel 1. Los encabezados (desde h1 a h6) son títulos que definen secciones implícitas en el documento y organizan el contenido en una estructura jerárquica.

Parámetro:

Align: Es la alineación horizontal del encabezado. Existen cuatro valores posibles (insensibles a mayúsculas/minúsculas):

1.Left: El encabezado se alinea a la margen izquierda.

2.Center: El encabezado es centrado.

3.Right: El encabezado se alinea a la margen derecha.

4.Justify: El encabezado es justificado o alineado a ambas márgenes.

Elemento p y /p:

Representa un párrafo. Se considera párrafo a un bloque de texto compuesto de una o más oraciones que se refieren a un tema en particular y que normalmente se separa de otros por un espacio en blanco.

Parámetros:

Align: Es la alineación horizontal del texto dentro de este elemento y de todos sus descendientes. Existen cuatro valores posibles (insensibles a mayúsculas/minúsculas):

1.Left: el texto se alinea a la margen izquierda.

2.Center: el texto es centrado.

3.Right: el texto se alinea a la margen derecha.

4.Justify: el texto es justificado o alineado a ambas márgenes.
